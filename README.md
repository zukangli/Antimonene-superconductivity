# Data and Code for Antimonene Superconductivity Study

This repository contains the data and computational input files for the manuscript:

"Strain-Tunable Superconductivity in Alkali-Metal Intercalated Bilayer Antimonene: A First-Principles Study"

## Contents

- structures/
   Pristine bilayer antimonene structures (VASP POSCAR format), serving as initial configurations before metal intercalation

- QE_inputs/
  Quantum ESPRESSO input files (relax, scf, scf.a2f, elph)

## Notes

These files are sufficient to reproduce the main results of the paper.
